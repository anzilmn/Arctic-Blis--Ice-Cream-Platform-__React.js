# Arctic Bliss - Premium Ice Cream Shop Website

A modern, sleek, and fully functional frontend application for a luxury ice cream brand, built with React.js and Tailwind CSS.

## ✨ Features

* **Responsive Modern UI**: Built with Tailwind CSS and Framer Motion for smooth animations and a premium feel.
* **Product Showcase**: Browse available flavors with interactive detail views.
* **Dynamic Cart System**: Add/remove items with instant toast notifications.
* **Full Checkout Flow**: Integrated simulated payment methods (Card, UPI, COD).
* **User Authentication**: Simulated login/signup system using `localStorage`.
* **Order Tracking**: View order history and track active orders with a visual progress timeline.



## 🛠️ Built With

* **React.js**
* **Tailwind CSS**
* **Framer Motion** (for animations)
* **Lucide React** (for icons)
* **React Router DOM** (for navigation)
* **React Hot Toast** (for notifications)
